# paper-harness

This directory is the **paper-harness** workflow. When the user requests paper analysis, summary, or study-note generation — recognized by any of:

- an arXiv URL (`https://arxiv.org/abs/...` or `/pdf/...`)
- a DOI (`10.xxxx/...`)
- a generic PDF URL
- a local `.pdf` path

— follow the workflow in `SKILL.md` in this directory. Read `SKILL.md` before doing any of the steps; treat it as authoritative.

For unrelated tasks in this directory (e.g., editing the scripts, debugging output, reviewing prompts), `SKILL.md` is reference material only — do not auto-invoke the workflow.
